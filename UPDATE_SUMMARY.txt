│                                                                               │

│    ├─ Tipo: Número entero                                                    │

│    ├─ Tipo: Texto libre                                                      │

│    informacion_capturada (dict, REQUERIDO)                                  │
│    ├─ Tipo: Diccionario de pares clave-valor                                │
│    ├─ Claves: info_reg_1, info_reg_2, info_reg_3, ... (del contexto)        │

│                                                                               │
│    seccion (str, OPCIONAL)                                                   │
│    ├─ Tipo: Texto libre                                                      │
│    ├─ Default: "etapa_actual"                                                │
│    └─ Ejemplo: "etapa_actual", "etapa_1", "informacion_personal"            │
